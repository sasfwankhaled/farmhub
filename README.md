# FarmHub - Farm Management System

FarmHub هو نظام متكامل لإدارة المزارع، تم تطويره باستخدام أحدث التقنيات لضمان الفعالية والسرعة في متابعة الإنتاج، العمال، والمصاريف.

## الميزات الرئيسية
*   **إدارة الإنتاج**: تتبع الشحنات (الطرود) من لحظة التحميل حتى البيع والأرشفة.
*   **إدارة العمال**: تسجيل الدوام (الحضور والانصراف) وحساب المستحقات المالية بشكل آلي.
*   **لوحة تحكم تحليلية**: تقارير مالية ورسوم بيانية توضح صافي الربح والمبيعات حسب المحصول.
*   **إدارة المصاريف**: تتبع كافة مصاريف المزرعة (مياه، كراتين، لوازم) وربطها بالمزارعين.

## التقنيات المستخدمة
*   **الواجهة الأمامية**: React.js مع TypeScript.
*   **التنسيق**: Tailwind CSS.
*   **المحرك الخلفي وقاعدة البيانات**: Supabase (PostgreSQL).
*   **التفاعل**: Framer Motion للأنيميشن والرسوم.
*   **الرسوم البيانية**: Recharts.

## هيكلية المشروع
*   `/frontend`: يحتوي على كود الواجهة الأمامية بالكامل.
*   `/database`: يحتوي على توثيق الدوال البرمجية (Functions) المستخدمة في Supabase.

## تعليمات التشغيل
1. قم بالدخول إلى مجلد `frontend`:
   ```bash
   cd frontend
   ```
2. قم بتثبيت التبعيات:
   ```bash
   npm install
   ```
3. قم بإنشاء ملف `.env` وقم بإضافة مفاتيح Supabase الخاصة بك:
   ```env
   VITE_SUPABASE_URL=your_url
   VITE_SUPABASE_ANON_KEY=your_key
   ```
4. تشغيل المشروع في وضع التطوير:
   ```bash
   npm run dev
   ```

## الرخص
هذا المشروع مرخص بموجب رخصة Apache-2.0.
